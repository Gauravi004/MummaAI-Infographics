IP Breakdown — Mumma AI
Copyright Act 1968 (Cth) applied
s.10 — Software = Literary Work
Mumma AI's source code is automatically protected under Australian Copyright Act 1968 the moment it was written. No registration needed.
s.35(1) — Who owns it?
Built independently on personal laptop — Gauravi owns the copyright, not Deakin University. Students are not employees.
Open Source Libraries
MIT — Flask ✓ MIT — Pytest ✓ Apache 2.0 — gTTS ⚠ GPL — check required !
Apache 2.0 libraries require attribution in documentation. GPL libraries require open-sourcing if commercialised.
